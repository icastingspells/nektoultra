# NektoULTRA — nekto.me
Форк расширения NektoPRO с обходом IP-банов.

## Возможности

- **Авто-скип** — автоматически пропускает собеседников с человекоподобными задержками (без бана)
- **Изменение голоса** — pitch-shifting и аудио-эффекты (реверб, бас-буст, радио)
- **Распознавание речи** — офлайн через Vosk (русская модель встроена), голосовой скип по словам-триггерам
- **Горячие клавиши** — `Alt+L` скип, `Alt+M` вкл/выкл микрофон (настраиваются)
- **Статистика** — счётчик бесед, общее время, рекорд, ежедневная статистика, streak
- **Блокировка трекеров** — вырезает Яндекс.Метрику, Google Analytics, Bugsnag

## Что изменено по сравнению с оригиналом

- Авто-скип теперь срабатывает через **1.2–3.8 секунды** рандомной задержки вместо мгновенного клика
- Голосовой триггер добавляет **400–1000ms** задержки перед скипом
- Весь цикл скипа (отмена → подтверждение → поиск) занимает **1.4–3.1 секунды** вместо 200ms
- Убран WebRTC DataChannel `nektopro-secret` из SDP — он был виден на сервере некто.me как fingerprint бота и вызывал бан
- Убрана функция детекции "собеседник использует NektoPRO" (была завязана на тот же DataChannel)

## Установка

1. Скачай репозиторий (или zip-архив релиза)
2. Открой `chrome://extensions/`
3. Включи **Режим разработчика**
4. Нажми **Загрузить распакованное расширение** и выбери папку проекта

После обновления файлов обязательно нажми 🔄 **Reload** на карточке расширения в `chrome://extensions/`.

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Alt+L` | Скип собеседника |
| `Alt+M` | Вкл / Выкл микрофон |

Клавиши можно переназначить в настройках расширения.
