# NektoULTRA — nekto.me
Форк расширения NektoPRO
Расширение для Chrome/Firefox, которое прокачивает аудиочат на [nekto.me](https://nekto.me).

## Возможности

- **Авто-скип** — автоматически пропускает собеседников по заданным критериям
- **Изменение голоса** — pitch-shifting и аудио-эффекты (реверб, бас-буст, радио)
- **Распознавание речи** — офлайн через Vosk (русская модель встроена)
- **Горячие клавиши** — `Alt+L` скип, `Alt+M` вкл/выкл микрофон (настраиваются)
- **Статистика** — счётчик бесед, общее время, рекорд, ежедневная статистика, streak
- **Блокировка трекеров** — вырезает Яндекс.Метрику, Google Analytics, Bugsnag
- **SoundPad интеграция** — воспроизведение звуков через uwupad.me

## Установка

1. Скачай репозиторий (или zip-архив релиза)
2. Открой `chrome://extensions/` (или `about:addons` в Firefox)
3. Включи **Режим разработчика**
4. Нажми **Загрузить распакованное расширение** и выбери папку проекта

## Файлы

| Файл | Описание |
|------|----------|
| `manifest.json` | Манифест расширения (MV3) |
| `background.js` | Service worker — проксирует запросы к uwupad.me и обрабатывает команды |
| `inject-bootstrap.js` | Content script — точка входа, внедряется на страницу audiochat |
| `nekto-pro-inject.js` | Основная логика: авто-скип, голосовые эффекты, панель управления, статистика |
| `vosk-lib.js` | Движок офлайн-распознавания речи (Vosk WASM) |
| `vosk-model-ru.tar.gz` | Русская языковая модель для Vosk |
| `dnr-uwupad.json` | Правила DeclarativeNetRequest — подменяет Referer для uwupad.me |

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Alt+L` | Скип собеседника |
| `Alt+M` | Вкл / Выкл микрофон |

Клавиши можно переназначить в настройках расширения.
